# Development

## Database

1. Clone Repositories
2. install yarn packages to run project with following command:
```
$ yarn install
```
3. Copy file ```.env.template``` y rename ```.env```
4. Settieng Environment varible
5. For Running DataBase

```
$ docker compose up -d
```
6. Running
```
$ yarn start:dev
```






## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
